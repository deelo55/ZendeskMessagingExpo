## Features

- 🔥 Zendesk Expo module
- 🗣️ Basic conversation features
- 🔔 Push Notifications (no native code needed!)
- ✅ Support SDK events
- 🔑 JWT Authentication
- 🚗 Visitor Path
- 📝 Conversation Metadata(fields and tags)

## Installation

```sh
npm install expo-zendesk-messaging
```
